Evana Workflow Optimization

Overview

Evana is a risk analysis company, specializing in delivering rapid, reliable risk assessments to businesses across various industries. The company's core platform enables clients to analyze complex data sets, assess potential risks, and make informed decisions with confidence. However, the existing system presented usability challenges that hindered efficiency.

Users struggled with disorganized data presentation, inefficient workflows, and a lack of automation, making risk assessments more time-consuming than necessary.

I led the UX design efforts to create a more intuitive and streamlined platform, improving data visualization, workflow automation, and accessibility to empower users in their decision-making processes.

Challenge

Risk assessment requires fast, data-driven insights. However, Evana's platform faced several barriers:

Complex data structures
Users found it difficult to extract relevant risk factors due to cluttered data presentation.

Inefficient workflows
The platform lacked a clear task-oriented flow, leading to frustration and longer processing times.

Limited automation
Users needed better predictive analytics and automated reporting to optimize their assessments.

Fragmented user experience
The interface did not provide a seamless experience across desktop and mobile devices, limiting accessibility.

The objective was to create a more intuitive, efficient, and responsive platform that enables risk analysts to generate insights faster and with greater accuracy.

Research

To design a solution, I applied a user centered research approach.

Stakeholder and User Interviews

  • Conducted interviews with stakeholders, risk analysts, and underwriters to understand their daily challenges.
  • Identified key pain points in navigating and analyzing risk data.

Usability Testing

  • Observed users interacting with the existing platform to identify usability issues and inefficiencies in workflows.
  • Evaluated user behaviors to determine the most critical decision-making touchpoints.

Competitive Analysis

  • Researched data storage visualization tools to identify best practices in document storage platforms.
  • Evaluated usability features, data visualization tools, and automation capabilities.

Prototyping & Iterative Testing

  • Created wireframes and interactive prototypes to test new design concepts and workflow efficiency with users.
  • Iteratively refined the design based on user feedback and usability testing results.

Solution

Through user research and design, I was able to create an intuitive and seamless user experience by prioritizing simple, clean user interfaces that would allow users to upload documents, view current and past accounts, and download Evana's analysis results.

Post-Design Implementation

After finalizing the design, I collaborated with developers and stakeholders to ensure smooth implementation. Key post-design activities included:

Developer Handoff & Collaboration
Provided detailed design documentation, component specifications, and interactive prototypes to facilitate seamless development.

Pilot Testing & Feedback Iteration
Conducted a beta rollout to a select group of analysts, gathering feedback and making necessary refinements.
Training & Onboarding
Created user guides and facilitated training sessions to ensure smooth adoption of the new system.

Performance Analysis
Monitored post-launch analytics to assess improvements in task completion time, accuracy, and user satisfaction.

Impact and Results

The redesigned Evana platform significantly enhanced efficiency and user experience. Key outcomes included:

30% Reduction in Analysis Time
Automated workflows and improved document visualization led to faster decision-making.

Streamlined Workflows
The new flow reduced processing times and minimized user errors.

Enhanced Automation
Automated reporting features enabled users to generate insights more quickly and accurately.

Conclusion

By leveraging user research, iterative design, and a collaborative implementation approach, I successfully transformed Evana's risk analysis platform into a more effective and user-friendly system. This project reinforced the value of aligning design decisions with user needs and business objectives, ultimately driving efficiency and accuracy for risk analysts.
